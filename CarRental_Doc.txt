"Car Rental" fullstack project :-

Technologies:
  Reactjs, Tailwindcss, "PrebuiltUI"(website gives tailwindcss components), Nodejs, Expressjs, Mongodb.  
  tools:  
      Google Fonts - 'Outfit' font
      Imagekit.io - images, 
      Vercel for deployment. 



Features :-
1) User dashboard
  login, search car, 
  my bookings,
  car details, pickup location details, date. 
  owner approval.

2) Owner dashboard  
   Edit profile, bookings page, manage cars,
   add car, details, price, available status, description. 
   payment, accept booking.
   monthly revenue.


-----------------------

App Structure:-
1) Home Page 
Hero Section
  - Navbar(Logo, menubar searchbar, login)
  - Hero Section(heading, car image, pickuplocation pickdate return date, searchbar)

Featured Vehicles Section 
  - Car(car type, each day money, car details)   
    event listener click on 'car card' takes to 'car full details' page.

Banner Section 
  - information, car photo

Testimonial Section
  - custom heading
  - Testimonial component('PrebuiltUI' tailwindcss library component)  

NewsLetter Section 
  - Newsletter component('PrebuiltUI' tailwindcss library component)  

Footer Section 
  - Footer component('PrebuiltUI' tailwindcss library component)  
    *** note: Footer must be present in all Pages. but, not in 'Owner Dashboard Page'.

2) Car Details Component  ex:http://localhost:5173/car-details/67ff5bc069c03d4e45f30b77
  left side:
   - Car image
   - Car description
   - Car features
  rightside:
   - Booking Form
  Footer section

3) Cars Page 
  - title 
  - search input to find Car by make/model/feautes
  - all Cars Cards 
  Footer section
    
4) My Bookings Page   
  - title 
  - booking cards (car image + booking details + price) 
  Footer section

5) Owner Dashboard Page 
  - Dashboard tab component  
  - Add Car tab component 
  - Manage Cars tab component 
  - Manage Bookings tab component
